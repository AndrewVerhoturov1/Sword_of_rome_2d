# Описание проекта

## Название проекта

`sword-of-rome-web`

## Тип проекта

`browser-based 2D authoring tool / editor / tabletop sandbox`

## Главная цель

Сделать веб-инструмент для создания, настройки, тестирования и дальнейшего розыгрыша варгеймов с картой, spaces, связями, фишками, сценариями и связанными данными. Sword of Rome-like модуль используется как первый тестовый модуль, а не как единственная цель проекта.

## Ключевые ограничения

- Публичный GitHub-репозиторий должен оставаться удобным для внешних чатов и навигации.
- `/v1` история должна быть видна в repo, а Kilo-runtime артефакты — нет.
- `arena-prototype-launcher/`, `_local/` и `output/Arena tests/` остаются локальными и по умолчанию не публикуются.
