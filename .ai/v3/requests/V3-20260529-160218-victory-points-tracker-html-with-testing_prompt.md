### Task Description

Создай V3 artifact package, который содержит ровно один новый project file:

`table-sandbox/v3-pilots/victory_points_tracker_russian.html`

Это standalone страница для учёта очков победы двух сторон в варгейме или настольной игре.

Что должно быть:
- отображение очков `Сторона А`;
- отображение очков `Сторона Б`;
- кнопки `+1` и `-1` для каждой стороны;
- кнопки `+5` и `-5` для каждой стороны;
- кнопка `Сброс`;
- строка `Ведёт` с вариантами:
  - `Ведёт: Сторона А`
  - `Ведёт: Сторона Б`
  - `Ведёт: Ничья`
- строка `Последнее изменение`;
- русский интерфейс;
- inline CSS;
- inline JavaScript;
- никаких внешних зависимостей.

Минимальная логика:
- старт: `Сторона А = 0`, `Сторона Б = 0`, `Ведёт: Ничья`;
- `+1` и `+5` увеличивают очки нужной стороны;
- `-1` и `-5` уменьшают очки нужной стороны, но итог не должен уходить ниже `0`;
- `Сброс` возвращает всё к начальному состоянию;
- `Последнее изменение` обновляется после каждого действия.

Визуально:
- аккуратный standalone layout;
- читаемые крупные числа;
- явное разделение сторон;
- нормальный вид на десктопе и в узком окне;
- без фальшивого 3D;
- с лёгким настольным/варгейм-настроением, но без перегруза.
