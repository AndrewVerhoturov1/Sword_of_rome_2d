# Human checks

Выполни эти проверки руками в браузере после импорта файла в репозиторий.

1. Открой:
   `table-sandbox/v3-pilots/battle_status_dashboard_russian.html`

2. Проверь стартовое состояние:
   - Раунд = 1
   - Фаза = `Инициатива`
   - Инициатива = `Сторона А`
   - Очки победы стороны А = 0
   - Очки победы стороны Б = 0
   - Очки командования стороны А = 0
   - Очки командования стороны Б = 0
   - `Ведёт по очкам: Ничья`
   - строка `Последнее событие` описывает стартовое состояние

3. Нажми `Следующая фаза` несколько раз.
   Проверь порядок:
   `Инициатива → Движение → Бой → Снабжение → Инициатива`.

4. Нажми `Предыдущая фаза` несколько раз.
   Проверь обратный круг:
   `Инициатива → Снабжение → Бой → Движение → Инициатива`.

5. Нажми `Следующий раунд`.
   Проверь, что раунд увеличился на 1, а фаза стала `Инициатива`.

6. Нажми `Сменить инициативу`.
   Проверь, что инициатива переключается между `Сторона А` и `Сторона Б`.

7. Для стороны А проверь:
   - `+1` и `−1` для очков победы;
   - `+1` и `−1` для очков командования.

8. Для стороны Б проверь:
   - `+1` и `−1` для очков победы;
   - `+1` и `−1` для очков командования.

9. На нулевом значении нажми `−1`.
   Проверь, что счётчик остаётся 0 и не становится отрицательным.

10. Проверь `Ведёт по очкам`:
    - если у стороны А больше очков победы — показывает `Сторона А`;
    - если у стороны Б больше очков победы — показывает `Сторона Б`;
    - если очки равны — показывает `Ничья`.

11. После разных действий смотри на `Последнее событие`.
    Проверь, что строка обновляется и понятно описывает последнее действие.

12. Нажми `Сброс`.
    Проверь возврат к стартовому состоянию.

13. Оцени визуально:
    - крупные значения читаются;
    - две стороны явно разделены;
    - общий статус хода понятен;
    - страница нормально выглядит на десктопе;
    - страница остаётся удобной в узком окне;
    - историческое настроение есть, но без перегруза и без фальшивого 3D.
